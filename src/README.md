src/

## Bibliotecas a serem instaladas

```
pandas

```

Para instalar as bibliotecas basta dar o comando

```bash
# Instalar dependências
pip install nome_biblioteca
```

## Como rodar a aplicação

```bash
# Rodar a aplicação
streamlit run src/app.py
```
